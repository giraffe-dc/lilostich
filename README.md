Створіть інтерактивний веб-сайт на основі сценарію дитячого квесту "Ліло та Стіч. Місія Охана", призначений для дітей віком 6-7 років. Сайт повинен імітувати структуру квесту з проектором, де користувачі (діти) проходять через 8 інтерактивних станцій, збираючи фрагменти кристала Охани (представлені як пелюстки гібіскуса). Тривалість віртуального проходження — близько 55-70 хвилин, з елементами анімації, звуку, ігор та взаємодії.

Тема та дизайн: Гавайський стиль з яскравими кольорами, пальмами, пляжами, елементами з мультфільму "Ліло та Стіч" (персонажі як Стіч, Ганту, Капібара). Фонова музика: гавайська (інтегрувати аудіо-файли або плейлисти для автоматичного відтворення з можливістю паузи). Використовуйте анімації (CSS/JS) для переходів між слайдами, ефектів (наприклад, вибухи, стрибки персонажів) та інтерактиву.
